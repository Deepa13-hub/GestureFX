# GestureFX Spark Edition

A real-time hand gesture visual effects project built using Python, OpenCV, and MediaPipe.

## Features

- Hand tracking
- White motion trail
- Spark lightning effect
- Chakra effect
- Two-hand finger connections
- Gesture-based mode switching

## Technologies Used

- Python
- OpenCV
- MediaPipe

## How to Run

Install dependencies:

```bash
pip install opencv-python mediapipe
```

Run:

```bash
python main.py
```

## Controls

- ✌️ Peace Sign → Spark Lightning
- ✊ Closed Fist → Chakra Mode
- 🖐️ Open Hand → Clear Trail
